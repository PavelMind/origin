#### Паттерн "прокси (заместитель)".
Класс CacheProxyDB содержит кэш для базы данных, имитируемой классом VeryHeavyDatabase. Если переданный ключ в метод GetData() есть в кэше, то она вернёт "Get from cache", иначе обратится к "реальной" базе данных.  
Метод GetData() а классе OneShotDB должен возвращать данные по ключу из реального объекта то количество раз, которое указано в конструкторе. Исчерпав лимит получения данных, метод GetData всегда возвращает строку "error".
