#### Паттерн "строитель".
Класс SqlSelectQueryBuilder, который можно использовать для построения простых SELECT-запросов к базе данных.
- Методы построения запроса AddColumn, AddFrom, AddWhere могут вызывать в любом порядке. При этом запрос строится корректно.
- Если метод AddColumn не был вызван, запрос должен начинаться с SELECT * .
- Для простоты строки в запросе не выделяются кавычками.
- Вызовы AddFrom должны перезаписывать название таблицы в классе.
- Методы AddWhere(const std::map\<std::string, std::string\>&) и AddColumns(const std::vector\<std::string\>&) позволяют добавить в запрос сразу несколько условий и полей соответственно.

Класс потомок AdvancedSqlSelectQueryBuilder добавляет версию AddWhere(), которое позволяет добавить в условие WHERE запроса дополнительные операторы "больше" и "меньше".
