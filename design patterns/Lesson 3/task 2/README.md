#### Паттерн "наблюдатель".
Наблюдаемый класс с методами:
- void warning(const std::string& message) const;
- void error(const std::string& message) const;
- void fatalError(const std::string& message) const;  

Этот объект:
- умеет добавлять указатели на наблюдателей,
- оповещает наблюдателей о вызове каждого из методов,
- корректно обрабатывает разрушение объектов-наблюдателей.  

В иерархии классов-наблюдателей нужно было реализовать:
- класс для работы с предупреждениями, который будет печатать сообщение в консоль;
- класс для работы с ошибками, который будет печатать сообщение в файл по указанному пути;
- класс для работы с фатальными ошибками, который будет печатать сообщение в консоль и в файл по указанному пути.  

В наблюдаемом классе используется функция `setString(std::string message)`, которая в зависимости от условной проблемы с сообщением message оповещает соответствующий тип наблюдателя.
