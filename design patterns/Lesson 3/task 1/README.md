#### Паттерн "команда".
Команды для системы логирования.  
Класс consolLogCommand представляет команду, которая осуществляет печатать сообщения message в консоль, а класс fileLogCommand печатает сообщение в файл по указанному пути.  
Функция print(LogCommand* command, std::string massage) будет выполнять команды.
