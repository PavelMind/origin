### Курсовой проект "Парсер INI-файлов"
В задании было необходимо реализовать класс парсера INI-файлов, который предоставляет для пользователя одну шаблонную функцию, позволяющую получить значение переменной в определённой секции.  
Работа с классом должна выглядеть так:
```
ini_parser parser("filename");
auto value = parser.get_value<int>("section.value")
```
Если в файле нет значения для этой переменной, то нужно бросить исключение. Если есть проблемы со считыванием файла или файл некорректный, то нужно сообщить об этом в исключении.  

Дополнительное задание:  
Если файл содержит некорректный синтаксис, нужно указать в исключении, на какой строке встретилась проблема.  
Если искомого значения переменной нет, нужно вывести подсказку для пользователя — названия других переменных из этой секции. Возможно, пользователь опечатался.

#### Что было сделано
Для выполнения дополнительной задачи был создан отдельный класс исключения `myexpect`. Программа выкидывает исключение этого класса, когда обнаруживает ошибку в синтаксисе файла .ini. При этом выводится сообщение и номер строки файла, где обнаружена эта ошибка.

Реализовано было 2 варианта класса парсера `ini_parser`, которые находятся в пространствах имён variant1 и variant2. В обоих вариантах парсер имеет шаблонный метод `T get_value<T>(std::string)`, в который передаётся строка с названием секции и переменной через точку.

Класс ini_parser в variant1 обрабатывает ini-файл при каждом запросе метода get_value().  
Класс ini_parser в variant2 при создании считывает 1 раз содержимое ini-файла во внутреннюю структуру и при запросе берёт значения из неё.

Методы в основном аналогичны у вариантов. 
- pairRequest() - определяет секцию и переменную в запросе.
- getSection(), getVar(), getValue() выделяют из строки файла названия секции, переменной и её значения соответственно. 
- skip() возвращает true, если строка не содержит ничего значимого. 
- printVariant() возвращает список с существующими переменными в секции. 
- Специализации `T readValue<T>(std::string&)` преобразуют прочитанное значение в типе строки в тип C++.
