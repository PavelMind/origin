#### Большие числа.  
Класс для работы с большими числами, которые не помещаются в стандартные типы данных. Для этого класса должны быть определенны методы:
1. конструктор перемещения;
2. перемещающий оператор присваивания;
3. оператор сложения двух больших чисел;
4. оператор умножения на число.  

Для реализации использовался динамический массив. Конструктор может принимать число целого типа или строку с числом.
