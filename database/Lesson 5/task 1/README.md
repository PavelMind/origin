#### Программа для управления клиентами на C++.
Нужно хранить персональную информацию о клиентах: имя, фамилия, email, телефон. Телефонов у каждого клиента может быть любое количество. Нужно разработать написать класс для управления данными со следующими методами:
- Метод, создающий структуру БД (таблицы) - create()
- Метод, позволяющий добавить нового клиента - addClient()
- Метод, позволяющий добавить телефон для существующего клиента - addPhone()
- Метод, позволяющий изменить данные о клиенте - updateClient()
- Метод, позволяющий удалить телефон у существующего клиента - deletePhone()
- Метод, позволяющий удалить существующего клиента - deleteClient()
- Метод, позволяющий найти клиента по его данным — имени, фамилии, email или телефону - get()  

Реализация использует библиотеку libpqxx для работы с PostgreSQL. Запросы писались на SQL.  
Дополнительно был реализован метод, очищающий таблицы - cleanedTable()
