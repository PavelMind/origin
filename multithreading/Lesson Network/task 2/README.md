#### Многопоточный TCP-сервер.
TCP-сервер, работающий в многопоточном режиме и использующий сокеты Беркли. Сервер умеет:
- устанавливать соединения с несколькими TCP-клиентами
- обрабатывать каждое соединение в отдельном потоке  

Обработка каждого соединения состоит в приеме тестовой строки от TCP-клиента и отправки ответной строки.
