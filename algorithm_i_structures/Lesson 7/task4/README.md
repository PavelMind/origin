#### Графы.  
В задачах 1, 2, 3 и 4 требовались реализации обхода в глубину, обхода в ширину, поиска циклов и поиска компонентов связности соответственно. В задаче собраны все эти функции.  
Граф представлен матрицей смежности размерностью N x N. Матрица смежности будет задаваться в текстовом файле в формате: первым числом в файле идёт количество вершин графа (N). Далее располагаются N строчек, каждая из которых содержит N чисел — 1 или 0.  
Реализовано:  
void deepRound(int); - обход в глубину.  
При посещении каждой следующей ячейки программа должна вывести на консоль индекс посещаемой ячейки. В аргументах передаётся индекс вершины, с которой начинается обход.  
void widthRound(int); - обход а ширину.  
При посещении каждой следующей ячейки программа должна вывести на консоль индекс посещаемой ячейки. В аргументах передаётся индекс вершины, с которой начинается обход.  
bool haveCycle();  
Возвращает true, если в графе есть циклы.  
int connectComponents(std::vector\<int\>&);  
Находит количество компонентов связности и выводит для каждой вершины номер компонента связности. В аргумент передаётся вектор, в который будет записан результат - номер компонента связности для каждой вершины, равной индексу +1. Возвращает число компонент.
