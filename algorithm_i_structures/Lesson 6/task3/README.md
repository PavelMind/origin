#### Упрощённый алгоритм Рабина-Карпа.  
Необходимо создать функцию, которая находит в предоставленной строке заданную подстроку с помощью упрощённого алгоритма Рабина-Карпа. Функция должна принимать на вход две строки: первая - в которой надо искать подстроку, а вторая будет этой подстрокой. Функция должна возвращать индекс найденной подстроки в строке.  
В реализации, если искомая подстрока больше той, где идёт поиск, возвращается -1; если подстрока не была найдена, то возвращается -2.

В данном упрощёном алгоритме на каждой итерации вычисляется хеш подстроки в строке, в которой ведётся поиск. При перемещении из хеша вычитается код первого символа в подстроке, и добавляется код символа, идущим следующим. После чего происходит сравнение с хешем искомой строкой.
